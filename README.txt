Portuguese:

Esse foi meu primeiro projeto em node.js seguindo o curso do Net Ninja, caso queira que funcione no seu computador, troque a variável .env_sample para .env e coloque o link do seu DATABASE.
Além é claro, de instalar todos os pacotes pendentes com o comando -npm install

English:

This was my first project on node.js following the Net Ninja course, if you want it to work on your computer, change the variable .env_sample to .env and put your DATABASE link.
In addition, of course, to install all pending packages with the -npm install command
